Documento de Exercício Prático – HTML e CSS

Conversão de Documento Word para HTML e CSS

Este documento foi elaborado com o objectivo de servir como base para um exercício prático. O aluno deverá converter todo o conteúdo aqui apresentado para uma página HTML, aplicando estilos CSS externos de forma correcta e organizada.

Este parágrafo contém uma linha abaixo castanha.

Elementos a considerar no exercício:

  1. Estrutura básica de um documento HTML
  2. Utilização correcta de títulos e parágrafos
  3. Aplicação de estilos CSS
  4. Organização do conteúdo

O documento HTML final deverá conter uma hiperligação funcional para um site à escolha do aluno: W3Schools

Imagem

Neste ponto, o aluno deverá inserir uma imagem à sua escolha no documento HTML. A imagem deverá apresentar um nível de transparência de 15% aplicado via CSS.

Tabela de Competências Técnicas

Competência Descrição Nível
HTML Estruturação de páginas web Intermédio
CSS Estilização e layout responsivo Intermédio
JavaScript Interacção e dinamismo Básico

A tabela contém no mínimo 3 linhas e 3 colunas. O cabeçalho tem cor verde claro e o texto boldado.

Folha de Estilo Externa (CSS)

Todo o estilo visual é definido num ficheiro externo chamado estilo.css, corretamente ligado ao documento HTML através da tag <link> na secção <head>.

Boas Práticas

  • Use nomes de ficheiros claros e em minúsculas
  • Comente o código se necessário

Alojamento no GitHub

Após concluir o exercício, o aluno deverá:

  • criar um novo repositório público no GitHub
  • enviar os ficheiros index.html e estilo.css devidamnete organizados respeitando boas praticas de densevolvimento.

No final do exercicio o aluno devera entregar o link do alojamento deverá ser entregue até 12 horas de Segunda-feira.
